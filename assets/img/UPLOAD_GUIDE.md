# Image upload guide

The website references the image files below. Upload your photos here with **these exact
filenames** (case-sensitive) and the site will pick them up automatically — Vercel
redeploys on each commit. Until a file is present, that spot shows a tidy labelled
placeholder, so nothing ever looks broken.

**How to upload on GitHub:** open the repo → `assets/img/` → **Add file → Upload files** →
drag the photos in → **Commit changes**. (Logos go in the `accreditations/` subfolder.)

### Notes
- **Format:** keep photos as `.jpg` and logos as `.png` (the names above). If a file
  must differ, tell me and I'll update the reference.
- **Size:** photos ideally ~1600–2000px wide; logos any size (they're scaled to fit a
  uniform white chip).
- The **team portraits** currently reuse one silhouette as a neutral placeholder — send
  real headshots (e.g. `team-lee.jpg`) whenever ready and I'll wire them in individually.
- The **contact map** is a live embedded Google Map (no upload needed).
